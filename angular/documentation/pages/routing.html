Routing

Espire comes with 2 types of main view, which is auth view & app view, each view has it own route tree located at src/app/routes

  • App View

    - along with Navigation, Header, Footer & etc, mainly serve for core application.

  • Auth View

    - did not comes with any template components, mainly serve for authentication such like login, register & or other pre login pages.
